# 若依后端运行说明

当前后端已整理到：

```text
D:\学校比赛项目\互联网+\数字中医\server\ruoyi-backend
```

`ruoyi-admin` 是后端启动入口，默认端口为 `8080`。本目录仍保持原若依模块结构，不做 Java 包名、模块名或数据库表结构重构。

## 推荐入口

优先回到总项目根目录启动：

```bat
cd /d "D:\学校比赛项目\互联网+\数字中医"
start-project.bat check
start-project.bat
```

根目录脚本会依次处理 Redis、后端编译、`ruoyi-admin` 启动和当前主用管理端 `apps/admin-web` 启动。

## 私有密钥配置

讯飞等后端私钥不要写进受版本控制的 `application.yml`。项目内已经约定：

```text
server\ruoyi-backend\config\application-secret.yml
```

这是本地私有文件，不提交 git。仓库里会保留模板：

```text
server\ruoyi-backend\config\application-secret.example.yml
```

首次使用时：

```bat
cd /d "D:\学校比赛项目\互联网+\数字中医"
copy server\ruoyi-backend\config\application-secret.example.yml server\ruoyi-backend\config\application-secret.yml
```

然后只在本机填写真实密钥。`scripts\start-all.bat` 启动后端时会自动附带这个私有文件。

## 手动启动后端

```bat
cd /d "D:\学校比赛项目\互联网+\数字中医\server\ruoyi-backend"
mvn -pl ruoyi-admin -am install -DskipTests
cd ruoyi-admin
mvn spring-boot:run -Dspring-boot.run.arguments=--spring.config.additional-location=optional:file:../config/application-secret.yml
```

## 数据库与 Redis

- 数据库名：`tcm_vsl`
- SQL 副本位置：`D:\学校比赛项目\互联网+\数字中医\database\sql`
- Redis 默认地址：`127.0.0.1:6379`
- Redis 默认脚本路径：`D:\Redis\redis-server.exe`

如果数据库账号、密码或 Redis 配置不同，请检查：

```text
server\ruoyi-backend\ruoyi-admin\src\main\resources\application-druid.yml
server\ruoyi-backend\ruoyi-admin\src\main\resources\application.yml
```

## 管理端说明

当前主用管理端不是本目录下的 `ruoyi-ui`，而是：

```text
D:\学校比赛项目\互联网+\数字中医\apps\admin-web
```

`ruoyi-ui` 已随后端源码保留，主要作为若依原始前端或历史参考。除非明确需要恢复若依 Vue2 前端，否则日常开发优先维护 `apps/admin-web`。

## webapi 扩展服务

`webapi/webapi` 已保留源码，但不是主链路必启服务。涉及语音、图像等扩展能力时，再检查并启动：

```bat
cd /d "D:\学校比赛项目\互联网+\数字中医\server\ruoyi-backend\webapi\webapi"
mvn spring-boot:run
```
